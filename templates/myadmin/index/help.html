Instructions to HALLO Data Portal Management


Welcome to HALLO Data Portal Management ! The management site provides four parts of functions:
  1. Users management through Link: User Management. Click User Management Instruction for detailed instruction.
  2. Add/Edit/Delete/Check marine acoustic artificial intelligence model through link: Model Management. Click Model Management Instruction for detailed instruction.
  3. Add/Edit/Delete/Check marine acoustic datasets through link: Datasets Metadata. Click Datasets Metadata Instruction for detailed instruction.
  4. Add/Edit/Delete archive files for each Datasets through the link in the dataset list. Click Datasets Archive Instruction for detailed instruction.
  5. Add/Edit/Delete annotation file for each Datasets through the link in the dataset list. Click Datasets Annotation Instruction for detailed instruction.
  6. View/Delete visitors' download history through link: Check Vistors' Download History. Click Download History Instruction for detailed instruction.

User Management


Manager could add new users or edit the existing users.
